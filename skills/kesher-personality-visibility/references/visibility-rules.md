# Personality Visibility Rules

## Public By Default

- Self-declared values and relationship intent.
- Public profile prompts, interests, lifestyle tags, age, coarse city/region, observance label when user chose it.

## Private By Default

- BFAS answers and raw scores.
- Generated personality summaries.
- Private taste profile and hidden recommender state.
- Explanation history and provenance logs.

## Mutual Consent Only

- Compatibility reflection.
- Owner-previewed share-card summaries.
- Communication notes intentionally shared with a specific recipient.

## System Only

- Audit logs, abuse signals, legal/safety retention records, hidden ranking features, model traces.

When uncertain, choose the stricter layer and add an explicit user control before exposing more.
