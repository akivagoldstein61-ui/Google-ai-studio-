# Pacing Coach Contract

## Allowed Signals

- Session length.
- Swipe velocity.
- Repeated passes or repeated dismissals.
- User-disabled/dismissed state.
- Coarse client context such as current discovery surface.

## Blocked Signals

- Private messages.
- Raw personality answers.
- Protected traits.
- Attractiveness/desirability.
- Exact location.
- Shame or scarcity-based behavioral targeting.

## UX Rules

The nudge must be dismissible, rate-limited, and non-blocking. It should invite reflection, not diagnose the user or pressure them to keep using the app.
