# Private Taste Contract

## Event Priority

1. Block/report/hard filter.
2. Explicit "more like this", "less like this", hide, direct edits.
3. Like, match, message sent/reply received metadata.
4. Profile open, prompt expansion, long dwell.
5. Impression-only events.

Implicit events are observations with uncertainty. Missing engagement is not dislike.

## Exclusions

Never use private message text, exact location, photos, protected traits, attractiveness/desirability, raw personality answers, or hidden safety records for taste learning.

## Reset

Reset learned taste vectors, explicit anchors, and cached explanation summaries. Preserve consent/legal/safety records only where policy requires.
