# Lovable Mobile/PWA Warning

Lovable output is web-first.

Do not claim native iOS, App Store readiness, production PWA, or Capacitor wrapper readiness unless the relevant tests have passed.

Required PWA tests: manifest, icons, service worker, install behavior, refresh/routing, mobile browser behavior.

Required Capacitor/iOS wrapper tests: auth, routing, images, safe areas, keyboard behavior, messaging, report/block, backend calls.

If tests are missing, say: Current status: mobile-friendly web. PWA/native wrapper claims are UNKNOWN until smoke-tested.
