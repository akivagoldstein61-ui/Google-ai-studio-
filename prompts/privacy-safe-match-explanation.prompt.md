# Privacy-Safe Match Explanation Contract

Trigger: Use for any match explanation, compatibility, coaching, prompt, or AI profile feature.

## Allowed input

Public profile fields explicitly approved for explanation, normalized overlap dimensions, high-level preference categories, consented user-visible information.

## Forbidden input

Raw questionnaire answers from another user, exact filters or dealbreakers, hidden ranking scores, private taste profile, moderation notes, admin annotations, sensitive inferred traits, raw safety reports.

## Output schema

Headline, safe rationale, shared dimensions, uncertainty note, no-guarantee statement, optional conversation prompt.

## Rules

Do not imply certainty, invent traits, reveal private or hidden signals, make attraction or chemistry scores public, reveal protected-trait inferences, or skip adversarial fixtures before release.
