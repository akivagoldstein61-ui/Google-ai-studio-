# Lovable Security View Remediation Prompt

Review Security View findings for Kesher.

Classify every finding: critical launch blocker, high launch blocker, medium fix before beta, low follow-up, false positive with explanation, or unknown needing manual review.

For each finding: affected area, likely cause, user/data risk, required fix, test required, owner, approval needed.

Security View is evidence, not final certification. Do not suppress critical/high findings without explicit human risk acceptance.
